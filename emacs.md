# Emacs Basics

## File / Buffer commands
|-----------------------|-----------------|-------------------------------------|
| NEW       Buffer      |     C-x b       |   <new buffer name>                 |
| SWITCH    Buffer      |     C-x b       |   <type/choose opened buffer name>  |
| OPEN      File        |     C-x C-f     |   <browse>                          |
| NEW       File        |     C-x C-f     |   <new file name>                   |
| SAVE      File        |     C-x C-s     |   <enter>                           |
|-----------------------|-----------------|-------------------------------------|
